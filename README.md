# Bike And Run 

## Fully functional EXAMPLE project written in Kotlin showing how you can calculate your calories,
edit your records, complete challenges, tracking on Google Maps and login/sign up to your Firebase account!

This project is an example that showing how to calculate your calories,
edit your records, complete challenges, tracking on Google Maps and login/sign up to your Firebase account.
Every part of this project is sample code which shows hot to do the following:

* Fill the inputs in calories bike/running fragment
* Open tracking, you MUST give a permission to enable tracking in Google Maps with API key
* Add or remove records with automatically database updating
* Check or uncheck challenges with automatically database updating
* Sign in or sign up to Firebase database with your email and password

## How to install this project

1. Clone this project
2. Install latest version of Android Studio https://developer.android.com/studio
   (Full setup https://www.youtube.com/watch?v=AGmoCiMNBFw&t=436s)
4. After you installed Android Studio you should wait a couple of minutes until it's build up your project
5. You can use my Map API key and Firebase API key, but please use yours
  (Setup for Google Maps https://www.youtube.com/watch?v=QVCNTPNy-vs and for Firebase https://www.youtube.com/watch?v=aiX8bMPX_t8)

## How to tweak this project for your own uses

I'd encourage you to clone and rename this project to use for your own purposes.
Please use yours Google Maps and Firebase API key. Instructions are above.

## Known issuses (Work in progress)

When you first time open a 'tracking' activity and give a permission to use gps location it won't work.
You must go back to main and return again to 'tracking'. Then it should works fine. 
Sometimes when you try to set a route in emulator to track, can stuck.
So, please close your emulator and try again.
